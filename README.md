# LogosEmpresa

Repositorio de logos empresariales para uso en proyectos web, plantillas HTML, correos corporativos y diseños internos.

## Descripción

Este repositorio contiene imágenes de logos organizadas para facilitar su reutilización en diferentes desarrollos, como:

- Plantillas de correo electrónico
- Firmas digitales
- Páginas web
- Landing pages
- Sistemas internos
- Documentación corporativa

Actualmente se incluye el logo de **COMSATEL**, el cual puede ser utilizado en diseños relacionados a la empresa.

## Archivos disponibles

| Archivo | Descripción |
|---|---|
| `Logo_Comsatel` | Logo oficial de COMSATEL en formato PNG con diversos estilos|

## Cómo usar el logo en HTML

Puedes usar el logo directamente en una plantilla HTML de esta manera:

```html
